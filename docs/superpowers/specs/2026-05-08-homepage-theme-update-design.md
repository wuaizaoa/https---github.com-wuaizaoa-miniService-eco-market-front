# 首页主题更新设计文档

**日期**：2026-05-08  
**主题**：环保主题 → 霓虹赛博科技主题  
**品牌**：微享商城

## 1. 设计概述

### 1.1 背景
原首页是环保主题（EcoMarket），现商城已更新为数码产品，需要将首页主题改为科技感的"霓虹赛博风"。

### 1.2 目标
- 更新品牌名称为"微享商城"
- 标语改为"微享商城，畅享科技生活"
- 整体视觉风格改为科技感/未来感
- 保持现有功能结构不变，只修改内容和样式

## 2. 详细设计

### 2.1 Hero 区域（主视觉）

**背景设计**：
- 渐变色：从 `#0a0a1a`（深黑）到 `#1a1a3a`（深蓝）
- 添加微妙的科技网格线图案
- 圆角保持 16px

**标题文字**：
- 主标题："微享商城" - 紫色霓虹发光效果
- 副标题："畅享科技生活" - 青色霓虹发光效果
- 段落文字：浅灰色，描述数码商城

**按钮**：
- 青色霓虹边框
- 悬停时发光效果
- 点击跳转到商品列表

**右侧装饰**：
- 科技设备 emoji 组合：📱💻🎧
- 字体大小：120px

### 2.2 商品展示区域

**推荐商品**：
- 标题："🌟 推荐商品"（保持不变）
- 文字颜色：调整为与深色背景协调
- 展示前4个商品

**新品上市**：
- 标题："🆕 新品上市"（保持不变）
- 展示后4个商品

### 2.3 底部特性展示

**特性卡片**：
- 背景：深色卡片
- 边框：微妙霓虹效果

**三个特性**：
1. **正品保障** ✅ - 确保每一件都是正品
2. **极速发货** ⚡ - 24小时内快速发货
3. **技术支持** 🔧 - 专业技术客服支持

### 2.4 颜色方案

## 3. 霓虹发光效果实现

### 3.1 文字发光效果
```css
text-shadow: 0 0 10px #722ED1, 0 0 20px #722ED1, 0 0 30px #722ED1;
```

### 3.2 按钮发光效果
```css
border: 2px solid #1890FF;
box-shadow: 0 0 10px #1890FF;
```

### 3.3 悬停效果
- 按钮悬停时发光增强
- 整体保持现代科技感
